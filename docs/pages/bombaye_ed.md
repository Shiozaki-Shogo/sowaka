# 護法少女ソワカちゃんED(電奇梵唄会Ver)
エンディング『極楽へ行きたいのなら』の一部修正版です。
護法少女ソワカちゃん電奇梵唄会『本朝OVER THE RAINBOW縁起』で上映されました。
一応これも「スピンオフ・番外編」に分類しておきます。
注釈ページはありませんが、最後のは[瀬戸内寂聴](http://ja.wikipedia.org/wiki/%E7%80%AC%E6%88%B8%E5%86%85%E5%AF%82%E8%81%B4)と『[キャプテンEO](http://jp.youtube.com/watch?v=AstW05bDiQU)』が元ネタでしょうかね？
