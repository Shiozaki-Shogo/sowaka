# 護法少女ソワカちゃん第６話・注釈

## 虚空蔵
[虚空蔵菩薩](http://ja.wikipedia.org/wiki/%E8%99%9A%E7%A9%BA%E8%94%B5%E8%8F%A9%E8%96%A9)は「虚空」（宇宙）に「蔵」せられた無限の智恵と慈悲を象徴する菩薩。
サンスクリット語では「アカシャガルバ」と呼ばれ、「アカシャ」は虚空、「ガルバ」は母胎を意味する。
曼荼羅では胎蔵界・虚空蔵院の中尊および釈迦院、金剛界・賢劫十六尊に配される。
空海上人は「虚空蔵求聞持法」を修することで、一切経を暗記したと伝えられている。

## アカシック・レコード
ドイツの神秘思想家・[ルドルフ・シュタイナー](http://ja.wikipedia.org/wiki/%E3%83%AB%E3%83%89%E3%83%AB%E3%83%95%E3%83%BB%E3%82%B7%E3%83%A5%E3%82%BF%E3%82%A4%E3%83%8A%E3%83%BC)が主張した「[アカシャ年代記](http://ja.wikipedia.org/wiki/%E3%82%A2%E3%82%AB%E3%82%B7%E3%83%83%E3%82%AF%E3%83%AC%E3%82%B3%E3%83%BC%E3%83%89)」の英語訳。
「アカシャ」はサンスクリット語で「空間」「宇宙」を意味する。「レコード」は「記録」のこと。
森羅万象・過去から未来まで全ての事象が宇宙のどこかに記録されているのではないか、という概念。
その記録にアクセスできれば、未来を予言できるとも考えられている。
また虚空蔵菩薩はアカシックレコードを擬人化したものではないか、という説がある。

## メッセージについて
頭文字や末尾、一定間隔の文字を利用して、一見した文章と異なる含意を隠蔽する手法は
和歌の[折句](http://ja.wikipedia.org/wiki/%E6%8A%98%E5%8F%A5)やカバラの[ノタリコン](http://ja.wikipedia.org/wiki/%E3%83%8E%E3%82%BF%E3%83%AA%E3%82%B3%E3%83%B3)から2chの縦読みまで古今東西に例がある。
[いろは歌](http://ja.wikipedia.org/wiki/%E3%81%84%E3%82%8D%E3%81%AF%E6%AD%8C)（これも空海上人作との伝承があり）の[暗号説](http://ja.wikipedia.org/wiki/%E3%81%84%E3%82%8D%E3%81%AF%E6%AD%8C#.E6.9A.97.E5.8F.B7.E8.AA.AC)もその一種。
