# 光あふれる食卓
くまみさんの『[【巡音ルカオリジナル曲】さなぎのおわり](http://www.nicovideo.jp/watch/sm6481657)』の曲にkihirohitoさんが歌詞をつけ、動画制作は背中P……というこれまでにない形式のコラボ作品です。

## 歌詞

**光あふれる食卓**

空を夕焼け雲が
ゆっくりと流れてく
目の前を子犬が
元気に駆けてゆく

遠くから僕のこと
たしかに呼んでる声がする
帰らなきゃ 今日の
晩ごはんは何かな

カレーを三日も続けて食べたら
やっぱり飽きてくる

朝の澄んだ空気と
冷たい水が目を覚ます
鳥たちのさえずりが
奏でるハーモニー

台所トントンと
ごはんの支度の音がする
朝食はふつう
あっさり目になるよね

カレーが五日も続けて出るのは
さすがに作りすぎ
