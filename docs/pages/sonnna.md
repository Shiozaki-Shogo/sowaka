# そんなところに神は宿らない
護法少女ソワカちゃんシリーズの作者・[kihirohito氏](http://piapro.jp/kihirohito)が[ピアプロ](http://piapro.jp/)で公開していた曲を、
VOCALOID2「[鏡音レン](http://www.crypton.co.jp/mp/pages/prod/vocaloid/cv02.jsp)」で再録した曲です。（[初音ミクver.はこちら](http://piapro.jp/a/content/?id=zy4wvkjwpeh67bdu)）
ソワカちゃんシリーズではありませんが、ソワカちゃんファンには外せない名曲ですのでご紹介することにしました。
曲名はドイツの建築家ミース・ファンデルローエの格言「神は細部に宿る」に由来すると思われます。
（この格言については美術史家・ヴァールブルクのものという説もあり）
ソワカちゃんシリーズとは一味違う[kihirohito氏](http://piapro.jp/kihirohito)の世界を堪能してください。
ちなみにまとめサイト管理人は↓ここの歌詞が大好きです。
> 今までのことまとめてみたけど　まとめないのとほぼ同じ
